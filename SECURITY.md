# Security Policy

Auto-CV handles sensitive personal data such as CVs, cover letters, job application history, and potentially Gmail credentials.

## Current Status

This project is in early concept and foundation stage.

## Reporting

If you notice a security issue in the public repository, open a GitHub issue without sharing secrets, tokens, private email content, or personal documents.

## Principles

- Do not commit credentials.
- Do not commit Gmail OAuth tokens.
- Do not commit generated personal documents.
- Do not expose the local remote server publicly by default.
- Keep personal data in local ignored directories.
